# Анализ рынка вакансий для аналитиков данных

**SQL-анализ спарсенных вакансий hh.ru** по запросам «аналитик данных» и «системный аналитик».

## Ключевые находки

- Зарплатная вилка рынка: в среднем **~110 тыс. («от») - 154 тыс. («до»)**, ориентир ~130 тыс. руб.
- Лидеры по вакансиям: Москва (1247), Санкт-Петербург (181); Нижний Новгород - в топ-5 регионов
- Обнаружены ошибки данных (зарплата «50 руб.») - задокументированы, исключены из выводов

## Что в коде ([analysis.sql](analysis.sql))

Агрегации, группировки, фильтрация, интерпретация результатов в комментариях к каждому запросу.
