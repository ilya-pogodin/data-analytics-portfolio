# Покупательская активность и монетизация игроков

**Проект курса «Аналитик данных» (Яндекс Практикум).** Игра «Секреты Тёмнолесья», free-to-play с внутриигровой валютой.

**Задача:** изучить влияние характеристик игроков на покупку внутриигровой валюты и оценить платёжную активность.

**Данные:** 1,3 млн транзакций, справочники игроков, рас и предметов (PostgreSQL).

## Ключевые находки

- Доля платящих игроков — **17,7%** (при среднем по индустрии 1–5%) — признак сильной монетизации
- Медиана чека (74,9) в 7 раз ниже среднего (525,7): много мелких покупок + редкие крупные
- **97,7% продаж** приходится на 2 предмета из 184 — экстремальный «длинный хвост»
- Найдены аномалии: 907 нулевых покупок, дубль предмета в справочнике

## Что в коде ([analysis.sql](analysis.sql))

Агрегатные функции, `PERCENTILE_CONT`, `FILTER`, многоуровневые CTE, оконные функции (`SUM() OVER`), `LEFT JOIN` с защитой от деления на ноль через `NULLIF`/`COALESCE`.
