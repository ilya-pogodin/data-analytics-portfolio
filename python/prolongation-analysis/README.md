# Анализ пролонгации проектов (Python, pandas)

**Задача:** оценить, насколько успешно аккаунт-менеджеры продлевают клиентские проекты, — рассчитать коэффициенты пролонгации K1/K2 помесячно по каждому менеджеру и отделу за 2022–2024 гг.

## Что сделано

- Очистка «грязных» финансовых данных: разные форматы месяцев, суммы с пробелами и запятыми, дубли, служебные статусы («стоп», «в ноль»)
- Преобразование широкого формата в длинный (`melt`), приведение дат
- Расчёт K1 (продление в следующий месяц) и K2 (продление после паузы) по каждому менеджеру
- Сводные таблицы (`pivot_table`), рейтинг менеджеров, динамика отдела
- Автоматический экспорт многостраничного Excel-отчёта

## Файлы

- [analysis.md](analysis.md) — код с пояснениями по шагам
- [report_anonymized.xlsx](report_anonymized.xlsx) — итоговый отчёт (дашборд, матрицы K1/K2, итоги года). Имена менеджеров обезличены, исходные данные не публикуются из соображений конфиденциальности.
